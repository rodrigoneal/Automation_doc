Automation Studio

Bem vindo ao Prod Studio. Vamos começar com a instalação e depois vamos cobrir com um Quickstart

Demos uma pincelada sobre o Prod Studio.

Caracteristicas

  • O Driver faz toda a orquestração, não precisando importar funções.
  • Não precisa ficar chamando o robot toda hora, basta passar o caminho do toml para o Driver.
  • O CLI grava um caminho relativo, facilitando colocar em produção em outro ambiente.
  • WaitElement espera o tempo settado.

Instalação

pip install git+https://github.com/digital-path/automation_studio#egg=automation_studio
