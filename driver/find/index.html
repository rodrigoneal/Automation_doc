Encontrando Elemento

Classe FindElement

A classe FindElement tem as funções para localizar o elemento na tela.

find_element

driver.find_element("elemento")

Procura o elemento na tela.

find_element(*paths: str)

paths: str: O nome dos caminhos das imagens/videos que ficam dentro do .toml.

Return: FireHiving: Objeto dá interatividade ao elemento.

first_to_find

Procura o elemento na tela e retorna o primeiro elemento que encontrar.

first_to_find(*paths: str, tip: List[int] = None)

paths: str: O nome dos caminhos das imagens/videos que ficam dentro do .toml.

tip: int: Dica na tela de onde fica a imagem/video procurado

Return: FireHiving: Objeto dá interatividade ao elemento.

implicitly_wait

Vai travar a execução do programa pelo tempo passado. É bem parecido com o sleep.

implicitly_wait(time: float)

time:float: tempo que vai travar a execução do programa.
